Research Computing Cloud Applications (RCC Apps)
==================================================
RCC Apps is a public git repository from Fluid Numerics that provides a good starting point for building HPC and Research Computing (RC) applications for Google Cloud Platform. The RCC Apps repository comes with recipes to "bake" VM images of common HPC/RC applications for Google Cloud and can provide a good starting point for creating VM images for your application. This documentation covers how to create, test, and deploy applications on Google Cloud
